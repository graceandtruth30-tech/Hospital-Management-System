# Project Report

## Overview
The Hospital Management System is a full-stack web application for managing patients, doctors, appointments, medical records, billing, and users.

## Architecture
- Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
- Backend: Node.js, Express.js
- Database: MySQL via Sequelize ORM
- Authentication: JWT and bcrypt

## Core Features
- Role-based access control
- CRUD operations for all core entities
- REST API endpoints
- Responsive login/register UI
- Dashboard for patient, doctor, and appointment overview

## Deliverables
- Source code, SQL scripts, README, API docs and deployment guide.
